# CNC G-Code Simulator

## Start
1. Speichere alle Dateien (`index.html`, `style.css`, `app.js`, `README.md`) in denselben Ordner.
2. Öffne `index.html` im Browser.

## Bedienung
- G-Code im linken Editor anpassen.
- `Parse` lädt/parst das Programm.
- `Start` startet die Simulation.
- `Pause` pausiert.
- `Step` führt genau eine Zeile aus.
- `Reset` setzt den Simulationszustand zurück.
- `Simulation Speed` steuert die Ausführungsgeschwindigkeit.
